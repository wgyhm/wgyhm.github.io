CF1495D BFS Trees

wgyhm
2023-11-03
最短路, 计数

Description

https://www.luogu.com.cn/problem/CF1495D

Solution

放任务计划里面好久的题目了。本来还以为是什么很困难的高妙题目。

$n,m$ 比较小，所以可以预处理一些东西以后对 $(s,t)$ 分别计算。

如果 $s,t$ 之间超过 $1$ 条最短路则一定无解。否则我们将路径上的点打上标记。

然后其实 $s,t$ 已经像缩成一个点了。我们只需要统计满足条件的父亲个数即可。像这题。
