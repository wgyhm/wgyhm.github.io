CF1806E Tree Master

wgyhm
2023-12-20
搜索, 根号分治

Description

https://www.luogu.com.cn/problem/CF1806E

Solution

水群的时候发现的很有意思的题，一眼只能想到树分块和莫队。

首先考虑根号分治，对深度是没有前途的，转而对每层节点数分治：

  • 当前层节点数 $x < \sqrt n$，总共有 $\dbinom{x}{2}=O(x^2)$ 种本质不同的询问，所以 $<\sqrt n $ 的总的本质不同的询问有 $\sum x_i^2\le (\sum x_i)\times (\max x_i )\le O(n\sqrt n)$。
  • 对于 $x\ge \sqrt n$，这样的层只有 $O(\sqrt n )$ 个，每次询问查询一次就是 $O(q\sqrt n)$。

为了避免本质相同的询问带来的影响，考虑记忆化搜索。只需要把节点数 $<\sqrt n$ 的记下来即可，可以避免使用哈希表或者 map。

复杂度 $O((n+q)\sqrt n)$。
