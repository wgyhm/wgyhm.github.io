P3350 [ZJOI2016] 旅行者

wgyhm
2023-08-24
分治

[ZJOI2016] 旅行者

Description

https://www.luogu.com.cn/problem/P3350

Solution

考虑分治。每次找当前矩阵较长的那一条边，取中点作为划分，暴力求出经过划分的线的最短路。如果两个点再划分之后的同一边，则递归下去。

每次都找中点划分，令 $S=nm$，则复杂度是 $S\sqrt S \log S$，后面的 $\log$ 是最短路。
