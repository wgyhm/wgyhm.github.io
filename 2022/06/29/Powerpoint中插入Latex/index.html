Powerpoint中插入Latex

wgyhm
2022-06-29
杂项

github link

使用要求

  • TeXLive或MiKTeX
  • GhostScript（如果最新版本出现问题，请尝试 gs9.26）
  • ImageMagick
  • （可选）TeX2img，用于通过 EMF 输出矢量图形（下载）。

下载并安装

  1. 从此存储库的发布页面或从IguanaTex 下载页面下载 .ppam 加载项文件。

  2. 加载加载项：在“文件”>“选项”>“加载项”>“管理：”中，选择“PowerPoint加载项”然后“转到…”，然后单击“添加新”，选择.ppam 文件在您下载它的文件夹中，然后“关闭”（如果您下载了 .pptm 源并将其保存为 .ppam，它将位于默认的加载项文件夹中）。

  3. 创建和设置临时文件夹：IguanaTex 需要访问具有读/写权限的文件夹来存储临时文件。

  4. 安装并设置 GhostScript , ImageMagick 和 TeX2img 的路径：

  • 在机器上的某个地方解压 TeX2img 后，运行 TeX2img.exe 一次

  • 在“主要设置”窗口中设置 gswin32c.exe 或 gswin64c.exe（注意“c”！）和 ImageMagick 的 magick.exe 以及 TeX2imgc.exe 的完整路径。

  • 仅需要通过 EMF 支持矢量图形（与 SVG 相比，优点：适用于所有 PowerPoint 版本，完全可修改的形状；缺点：某些显示器随机出现失真）
