安装 Devc++

wgyhm
2022-08-18
杂项

简单记录一下配置 devc++。可能是主要给初学的人看的。

Step 1 安装 devc++

网上随便找一个版本高一点的就好了

Step 2 修改 mingv64

众所周知，devc++ 已经很久没有更新了。而 github 上的 Embarcadero Dev-Cpp 个人认为不是很好用。为了更新 g++/gcc 的版本。我们要下载新的编译器。

到 mingw 官网上下一个8.2.0 或者更新的。

如果不想找，百度网盘link，提取码 g2cj

替换 dev 中的 mingv 即可。

Step 3 设置环境变量

找到 include 和 lib 文件夹加入系统变量，bin 文件夹加入 path。

保存后开 cmd ，输入 g++ -v 判断是否加入成功。
